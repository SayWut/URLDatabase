URL Database

קצת על האפליקציה

URL Database מאפשרת לך לשמור את האתרים המועדפים עליך במקום אחד.
כמו בכל מערכת של Server ו- Client על המשתמש ליצור חשבון משלו במערכת.
לאחר יצירת החשבון יפתח למשתמש אזור משלו במערכת אשר שם ישמר כל המידע של אותו משתמש
באפליקציה קיים Protocol בעל 6 חלקים

  1. Register
  2. Login
  3. ReceiveUserData
  4. AddUserItem
  5. RemoveUserItem
  6. Disconnected
  7. Close

האפליקציה ניתן לעשות את הדברים הבאים:

  • הרשמה וכניסה למערכת
  • הוספת קישורים לרשימה ומחיקתם
  • הצגת הקישורים בדפדפן בלחיצת כפתור
